### Spec

sqlite3 3.9.1
Node.js 0.12.7

### Usage
```sh
$ mkdir -p ~/projects # change your any directory
$ git clone https://github.com/kiyotan/vagrant-boxes.git ~/projects/rails_study
$ cd ~/projects/rails_study
$ vagrant up
$ vagrant ssh
```
