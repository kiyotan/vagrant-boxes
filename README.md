### Spec
centos67-x86_64 minimal with ...  
Ruby 2.2.1  
Rails 4.2.1  
sqlite3 3.9.1  
Node.js 0.12.7  

### Usage
```sh
# you should change below "~/project" to your directories
$ mkdir -p ~/projects
$ git clone https://github.com/kiyotan/vagrant-boxes.git ~/projects/rails_study
$ cd ~/projects/rails_study
$ vagrant up
$ vagrant ssh
```
